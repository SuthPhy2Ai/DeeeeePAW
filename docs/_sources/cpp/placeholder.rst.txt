步骤 1: 启动 C++ 推理引擎
^^^^^^^^^^^^^^^^^^^^^^^^^^

通过命令行启动 DeePAW C++ 推理引擎，加载模型权重并初始化 GPU 计算环境:

   :width: 100%
   :alt: 启动 DeePAW C++ 推理引擎

   命令行启动 DeePAW C++ 推理引擎

步骤 2: 执行预测任务
^^^^^^^^^^^^^^^^^^^^^

通过 Python 命令调用引擎，指定输入结构和参数，完成电荷密度预测:

   :alt: Python 命令完成预测任务
